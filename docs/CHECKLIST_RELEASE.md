# Checklist de release

## Código

- [ ] JavaScript aprovado com `node --check`;
- [ ] JSON válido;
- [ ] referências locais existentes;
- [ ] sem erros no console;
- [ ] componentes reutilizados quando possível;
- [ ] versão atualizada na interface e documentação.

## Funcional

- [ ] Salão abre;
- [ ] Hall calcula e renderiza;
- [ ] Buscar encontra membros;
- [ ] Registro ordena e filtra;
- [ ] Raid consulta composições;
- [ ] Galeria carrega;
- [ ] Liga cria e restaura torneios;
- [ ] downloads de canvas funcionam.

## Responsividade

- [ ] desktop;
- [ ] notebook;
- [ ] mobile;
- [ ] ausência de overflow inesperado.

## Documentação

- [ ] CHANGELOG atualizado;
- [ ] relatório da versão criado;
- [ ] resultados de testes salvos;
- [ ] evidências organizadas;
- [ ] ZIP testado.

## Maintenance Edition

- [ ] manifesto do núcleo funcional validado;
- [ ] documentos de manutenção e regras atualizados;
- [ ] checklist de nova raid testado;
- [ ] instruções Git/Cloudflare revisadas;
- [ ] `tools/run_tests_v7_6_1.sh` aprovado;
- [ ] pacote extraído em pasta limpa e validado.
