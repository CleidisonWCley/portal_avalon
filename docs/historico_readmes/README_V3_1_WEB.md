# Raid Hall v3.1 Web

Entrega de refinamento do front-end do Avalon Raid Hall.

## Escopo entregue

1. Separação do site por páginas HTML.
2. Refinamento visual medieval/fantasia com molduras e cards mais nobres.
3. Tabela geral sem insígnias, focada em desempenho atual/anterior.
4. Registro da Batalha simplificado para leitura operacional.
5. Animações leves de entrada e hover.
6. Integração dos mascotes `cley.png` e `olimpio.png`.
7. Limpeza de fundos indesejados nos assets visuais principais.
8. Melhorias de responsividade e legibilidade.

```powershell
cd web
python -m http.server 5500
```
