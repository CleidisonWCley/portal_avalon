# Portal Avalon V4 — Liga Reforjada e Hall Justo

## Destaques da versão

- Liga Avalon com navegação por fases/slides.
- Mapa sorteado por fase, não por card individual.
- Escolha de vencedor clicando diretamente no nome do jogador/time.
- Confirmação antes de avançar o vencedor.
- Final, disputa de bronze e pódio com troféus personalizados.
- Botão para copiar resultado.
- Botão para baixar imagem do pódio.
- Hall da Evolução blindado com regra de comparativo válido.
- Presença mínima de 15/21 para disputar o Hall da Evolução.
- Retorno à Batalha tratado como status separado.
- Ranking atual separado do ranking de evolução.

## Observação importante

A pasta `ocr/` foi mantida sem alterações.
