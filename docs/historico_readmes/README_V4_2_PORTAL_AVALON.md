# Portal Avalon V4.2 — Liga de Equipes e Mascotes do Portal

## Entregas principais

- OCR mantido sem alterações.
- Correção visual do Salão para evitar que a plataforma/piso atravesse o card da guilda.
- Integração dos mascotes contextuais nos títulos das páginas:
  - Salão
  - Hall
  - Buscar Ficha
  - Galeria
  - Liga
- Registro preservado com os mascotes originais já aprovados.
- CSS reutilizável para cabeçalhos com Cley à esquerda, título ao centro e Olimpio à direita.
- Ajustes de responsividade para ocultar/reduzir mascotes em telas menores.

## Liga Avalon

A Liga foi reorganizada no fluxo:

1. Seleção dos participantes.
2. Resumo da Liga.
3. Chaves da Liga.
4. Pódio.

### Melhorias para modo equipe

- Mantido sorteio automático de equipes.
- Adicionada definição manual de equipes.
- Equipes podem ser montadas a partir dos participantes confirmados.
- Equipes incompletas entram como reserva.
- Mantido fallback 2v2 adaptado quando o número de participantes não fecha bem para 3v3.
- Pódio de equipes exibe número da equipe e seus membros.
- Resultado copiado também considera membros das equipes.
- Imagem do pódio e card do campeão incluem informações de equipe quando aplicável.

## Compartilhamento

- Mantido botão de baixar imagem do pódio completo.
- Adicionado botão de baixar card do vencedor/equipe vencedora.
- Mantido botão de copiar resultado em texto.
