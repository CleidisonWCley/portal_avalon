# RELATÓRIO DE ATUALIZAÇÃO — PORTAL AVALON V7.0.3

## Versão

**Portal Avalon V7.0.3 — Refinamento Visual Final da Aba Raid**

---

## Objetivo

A V7.0.3 teve como foco refinar exclusivamente a aba **Raid**, melhorando a seleção de elementos, a organização dos estados da consulta e a apresentação dos cards de resultado no padrão visual do Portal Avalon.

## Melhorias implementadas

### 1. Combobox customizado de elementos

- O seletor de elementos deixou de ser uma área de chips fixa.
- Foi criado um combobox próprio para elementos, visualmente compatível com o combobox de boss.
- O campo mostra “Escolha um boss primeiro” enquanto nenhum boss foi selecionado.
- Após selecionar o boss, o combobox de elementos lista apenas os elementos disponíveis para ele.
- Elementos com opção única são selecionados automaticamente e continuam visíveis no padrão do novo componente.

### 2. Ícones coloridos dos elementos

- Os elementos passam a usar ícones coloridos presentes em:

```text
web/assets/img/elements/fire.png
web/assets/img/elements/water.png
web/assets/img/elements/earth.png
web/assets/img/elements/light.png
web/assets/img/elements/dark.png
web/assets/img/elements/basic.png
```

- Os ícones aparecem no combobox de elemento e também no cabeçalho dos cards de resultado.
- Foram adicionadas classes visuais por elemento: `element-fire`, `element-water`, `element-earth`, `element-light`, `element-dark` e `element-basic`.

### 3. Resultados apenas após consulta real

- A seção de resultados começa oculta.
- O card grande “Composições encontradas” não aparece mais antes de uma consulta.
- Ao trocar boss ou elemento, os resultados antigos são ocultados até uma nova busca ser feita.
- A área de resultados é exibida somente durante carregamento, erro, sem resultado ou resultado encontrado após a ação de busca.

### 4. Estados em cards padrão Avalon

- Foram criados cards estilizados para os estados:

```text
Carregando
Sucesso
Erro
Sem resultado
```

- O card de carregamento informa que o portal está buscando composições na GTales.top API.
- O card de sucesso aparece acima das composições encontradas.
- O card de erro informa que a fonte externa está temporariamente indisponível.
- O card de sem resultado orienta testar outra combinação de boss e elemento.

### 5. Cards de composição mais centralizados

- Os cards de composição passaram a ser renderizados dentro de uma lista centralizada.
- A largura dos resultados foi controlada para aproximar o visual do padrão usado na Liga Avalon.
- O cabeçalho de cada composição agora destaca o boss e mostra o elemento com ícone colorido.
