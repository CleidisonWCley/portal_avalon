# RELATÓRIO DE ATUALIZAÇÃO — PORTAL AVALON V7.0.2

## Versão

**Portal Avalon V7.0.2 — Combobox da Raid, Tradução PT-BR e Correção dos Avatares**

---

## Objetivo

A atualização V7.0.2 teve como foco refinar a aba **Raid** do Portal Avalon em três frentes principais:

1. Substituir o select nativo de Boss por um combobox customizado com busca.
2. Traduzir as observações vindas da GTales.top API para PT-BR de forma local e segura.
3. Investigar e reforçar o CSS/JS dos avatares dos heróis para corrigir o comportamento instável em tela cheia.

## Melhorias implementadas

### 1. Combobox customizado de Boss

- Select nativo de boss substituído por componente visual próprio.
- Campo com botão estilizado, busca interna e lista customizada.
- Lista de bosses com miniatura, nome amigável e elementos disponíveis.
- Filtro por texto em tempo real.
- Fechamento ao clicar fora e ao pressionar ESC.
- Estado de boss selecionado preservado no fluxo da consulta.

### 2. Chips de Elemento

- O seletor de elemento foi transformado em chips clicáveis.
- Somente elementos disponíveis para o boss selecionado são exibidos.
- Elemento selecionado recebe destaque visual.
- O botão “Buscar times” só libera após boss e elemento estarem selecionados.

### 3. Tradução PT-BR das observações

- O campo `infos` da API passa por limpeza de HTML.
- Tags como `<br>` são convertidas em quebras de linha.
- Frases comuns em inglês recebem tradução local para PT-BR.
- Termos técnicos importantes são preservados para não distorcer o sentido da estratégia.
- Observações vazias continuam ocultas.

### 4. Correção profunda dos avatares dos heróis

- A função de carregamento de imagem agora trata `load` e `error` separadamente.
- Quando a imagem carrega, o placeholder é ocultado e o container recebe estado `has-image`.
- Quando a imagem falha, o fallback é exibido sem quebrar o layout.
- A ordem dos caminhos dos assets de herói foi reorganizada para reduzir atraso e falhas visuais.
- O CSS de avatar foi reforçado com dimensões fixas, `position`, `z-index`, `object-fit`, `contain: paint` e overrides específicos para evitar conflito entre breakpoints.
