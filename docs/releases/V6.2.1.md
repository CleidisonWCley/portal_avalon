# Relatório — Portal Avalon V6.2.1
## Refinos dos Cards e Histórico Inteligente de Busca

## Resumo

Esta atualização refinou os canvas gerados na versão V6.2 e melhorou a busca da Ficha do Guardião com histórico inteligente local.

## Alterações realizadas

### Liga Avalon

- O card/canvas do modo 1v1 foi preservado.
- O canvas 1v5 foi ajustado para organizar participantes horizontalmente.
- A listagem vertical longa foi substituída por linhas compactas com separador `•`.
- O texto auxiliar **Top 3 avançam para a decisão** foi melhor posicionado.
- O canvas **Final Survival** foi ajustado para reduzir cortes e sobreposição de nomes.
- O canvas 3v3 foi ajustado para exibir membros das equipes em linhas mais compactas.
- Confrontos com **Aguardando** foram refinados visualmente.
- O texto **Classificado: Equipe X** foi mantido, mas com menor impacto visual.

### Buscar Ficha

- O texto **Portal Avalon** no canvas da ficha foi deslocado levemente para baixo, aumentando o respiro visual em relação à logo.
- A ficha manteve a estrutura visual aprovada da V6.2.
- A tela continua usando thumbs otimizadas das patentes.
- O canvas continua usando imagens rank em alta resolução.

### Histórico inteligente da busca

- As sugestões genéricas fixas foram removidas.
- A busca agora salva apenas nomes encontrados com sucesso.
- O histórico é armazenado em `localStorage`.
- A chave utilizada é `portal_avalon_busca_guardiao_historico`.
- Sugestões são exibidas do mais recente para o mais antigo.
- Nomes duplicados são evitados.
- O botão **Limpar** também apaga o histórico de sugestões.
