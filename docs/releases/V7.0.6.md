# RELATÓRIO — PORTAL AVALON V7.0.6
## Polimento Final da Experiência Raid

## Objetivo
A versão V7.0.6 refinou a experiência final da aba Raid, com foco em reduzir redundâncias visuais, transformar cartas em chips legíveis, adicionar feedback flutuante durante a busca e facilitar o retorno ao topo da página durante consultas longas.

## Arquivos alterados

```text
web/pages/raid.html
web/assets/js/raid.js
web/assets/css/styles.css
READM_AVALON_VF_1.md
```

## Atualizações realizadas

### 1. Cartas em chips visuais
A área de cartas dos heróis deixou de exibir apenas códigos técnicos como `skill-skill`, `crit-atk7` e `crit-crit`.

Agora as cartas são exibidas como chips visuais em português:

```text
skill-skill  →  Habilidade - Habilidade
crit-atk7    →  Crítico - Ataque
crit-crit    →  Crítico - Crítico
atk7-atk7    →  Ataque - Ataque
```

Também foi adicionada regra de fallback para códigos desconhecidos.

### 2. Toast/modal flutuante para busca
A busca de composições agora usa um aviso flutuante centralizado no padrão Avalon.

Estados contemplados:

```text
Consultando estratégias
Composições encontradas
Nenhuma composição encontrada
Fonte temporariamente indisponível
```

O toast não ocupa espaço fixo dentro da área de filtros.

### 3. Redução de redundâncias
A área de resultados agora mostra apenas o bloco real de resultados:

```text
RESULTADOS
Composições Encontradas
X composição(ões) encontradas para Boss • Elemento.
```

O card redundante de sucesso dentro dos resultados foi removido.

### 4. Fluxo de consulta preservado
Os status contextuais de seleção foram mantidos:

```text
Boss selecionado
Elemento selecionado
```

Mas o feedback de carregamento, sucesso, erro e vazio foi movido para o toast flutuante.

### 5. Auto-scroll preservado
Após encontrar resultados, a página continua rolando automaticamente até a Composição #1.

### 6. Botão “Voltar ao topo”
Foi adicionado um botão flutuante:

```text
↑ Topo
```

Ele aparece após o usuário rolar a página e leva de volta ao cabeçalho da aba Raid.

### 7. Skills p/ Stun preservado
A nomenclatura atual foi preservada:

```text
Skills p/ Stun
```
